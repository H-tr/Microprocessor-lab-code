Detailed Description

Provide two FIFO queues.

Provide functions that initialize a FIFO, put data in, get data out, and return the current size.

Remarks
The sizes of the FIFO must be a power of two

◆ RxFifo0_Get()

int RxFifo0_Get ( char *  datapt)

Get into RxFifo0.

Remove one 8-bit element from RxFifo0.

Returns the oldest value, first in first out

Warning
RxFifo0_Get itself need not be reentrant, but RxFifo0_Get must be thread-safe with RxFifo0_Put
Parameters
dataptpointer to a 8-bit storage location, into which to store the removed data
Returns
FIFOSUCCESS if ok, FIFOFAIL if empty and data could not be removed
