void TimerCapture_Init ( void(*)(uint16_t time)  task)

Initialize Timer A2.

Initialize Timer A2 in edge time mode to request interrupts on both edges of P5.6 (TA2CCP1). The interrupt service routine acknowledges the interrupt and calls a user function.

Parameters
taskis a pointer to a user function called when edge occurs
parameter is 16-bit up-counting timer value when edge occurred (units of 0.083 usec)
Returns
none
Note
Assumes low-speed subsystem master clock is 12 MHz
