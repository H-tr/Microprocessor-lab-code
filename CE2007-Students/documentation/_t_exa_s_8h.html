◆ TExaS_Init()

void TExaS_Init ( enum TExaSmode  mode)

Initialize logic analyzer or scope.

Initialize scope or logic analyzer, triggered by periodic timer

Parameters
modeis SCOPE or LOGICANALYZER
Returns
none
Note
This module only runs at 48 MHz
Uses UART0, EUSCI_A0 to pass data to PC
Uses ADC J3.26/P4.4/A9 in scope mode
Uses Timer32 periodic interrupts for 10 kHz sampling
Warning
TExaS_Init must be called before calling any other TExaS functions

Virtual 7-bit logic analyzer or 8-bit scope sampling J3.26/P4.4/A9
