Detailed Description

Provide receive/transmit functions for EUSCI A0.

EUSCI_A0 UART is connected to the PC via the USB cable
Edit FIFO0.h to change FIFO sizes

Remarks
UCA0RXD (VCP receive) connected to P1.2
UCA0TXD (VCP transmit) connected to P1.3
Interrupt driven device driver for the EUSCI A0 UART

Receive a string into MSP432.

Receive an ASCII string from EUSCI_A0 UART

Accepts ASCII input until <enter>

It echoes each character as it is input.

Warning
If more than max-1 characters are received, subsequent characters are discarded until the <enter> is received
Parameters
[out]bufPtis a pointer to an empty buffer into which characters are stored
[in]maxis the size of the buffer
Returns
32-bit unsigned number that was received
Note
EUSCIA0_Init must be called once prior
Backspace will remove last digit typed
