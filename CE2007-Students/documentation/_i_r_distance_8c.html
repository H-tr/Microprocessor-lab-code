Detailed Description

Take infrared distance measurements.

Provide mid-level functions that initialize ports and take infrared distance sensor measurements using the GP2Y0A21YK0F infrared distance sensors.

int32_t CenterConvert ( int32_t  nc)

Convert center infrared distance measurement.

Convert ADC sample into distance for the GP2Y0A21YK0F infrared distance sensor. Conversion uses a calibration formula
Dc = Ac/(nc + Bc) + Cc

Parameters
ncis the 14-bit ADC sample 0 to 16383
Returns
distance from robot center to center wall (units mm)
