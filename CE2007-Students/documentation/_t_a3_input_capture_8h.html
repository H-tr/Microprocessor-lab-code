Detailed Description

Initialize Timer A3.

Use Timer A3 in capture mode to request interrupts on rising edges of P10.4 (TA3CCP0) and P8.2 (TA3CCP2) and call user functions.

Initialize Timer A3.

Initialize Timer A3 in edge time mode to request interrupts on the rising edges of P10.4 (TA3CCP0) and P8.2 (TA3CCP2). The interrupt service routines acknowledge the interrupt and call a user function.

Parameters
task0is a pointer to a user function called when P10.4 (TA3CCP0) edge occurs
parameter is 16-bit up-counting timer value when P10.4 (TA3CCP0) edge occurred (units of 0.083 usec)
task2is a pointer to a user function called when P8.2 (TA3CCP2) edge occurs
parameter is 16-bit up-counting timer value when P8.2 (TA3CCP2) edge occurred (units of 0.083 usec)
Returns
none
Note
Assumes low-speed subsystem master clock is 12 MHz
