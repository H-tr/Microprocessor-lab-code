Activate Timer A0 interrupts to run user task periodically

Parameters

periodin 2us units (24/SMCLK), 16 bits
Returns
none
Note
Assumes low-speed subsystem master clock is 12 MHz. With divde by 24 the timer clock will be 500 kHz timer clock. The slowest period is 65535*2us=130ms, or about 8 Hz
